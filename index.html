如何接入你自己的项目

  1. 把你的部署脚本放到一个公开可访问的地址（GitHub Raw、对象存储、自己的服务器都行），替换代码里的 SCRIPT_URL。
  2. 把每个表单项对应到脚本里真正读取的环境变量名；脚本里建议都用 ${VAR:-默认值} 兜底，避免变量为空时报错。
  3. 这个文件本身就是一个可直接发布的静态页面：当作 index.html 扔进 GitHub Pages / Cloudflare Pages / Vercel 即可，不需要任何后端。
